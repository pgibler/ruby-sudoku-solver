Overview

Application is the engine of the sudoku solver. It uses a Provider to retrieve a BoardState and a Solver to configure the BoardState as close to a solution as possible. If the solution is found from the Application.run! method, a triple of (Provider, BoardState, Boolean) It declares a DEFAULT_PROVIDER and DEFAULT_SOLVER, both of which can be configured separately when the Application.run! function is invoked.
